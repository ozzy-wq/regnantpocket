# RegnantX

Production-style **Next.js 14 + Tailwind CSS + TypeScript** demo trading platform.

## Demo-only safety
- No real money
- No withdrawals
- No real trade execution
- No payment systems

## Run locally
```bash
npm install
npm run dev
```

## Test
```bash
npm run test
```

## Deploy
Vercel-ready (default Next.js setup).
